# 🏠 FindRoom - Direct PG & Room Finder

### 🚀 Problem Statement
Students and bachelors struggle to find affordable rooms and PGs without paying heavy brokerage fees. On the other hand, house owners face difficulties in finding reliable tenants.

### ✨ Our Solution
**FindRoom** is a web platform that directly connects room owners with students/bachelors. 
- **Owners** can seamlessly list their rooms with rent details, location, and facilities.
- **Tenants/Students** can browse available rooms dynamically in real-time and delete listings if needed.

### 🛠️ Tech Stack Used
- **Frontend:** HTML5, CSS3, JavaScript (Fetch API)
- **Backend:** Node.js, Express.js
- **Database:** MySQL
